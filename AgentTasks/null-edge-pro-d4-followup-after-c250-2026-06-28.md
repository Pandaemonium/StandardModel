# Message to Pro: D4 follow-up after C250 Aristotle partial result

Date: 2026-06-28

We are working on the null-edge Standard Model program. The broad goal is to
describe Standard Model fermions and interactions using discrete lightlike
steps, inspired partly by the Feynman checkerboard, while matching known
continuum physics in the appropriate limit.

The current hard target is Gate C1: produce a chiral fermion release without
unwanted light mirror branches. We have pivoted away from treating the finite
retarded null-edge seed itself as the release operator. The current mainline is
a null-edge overlap/Ginsparg-Wilson architecture:

```text
H_tet(k) = gamma5 * [ (i/a) sum_A B_A sin(k_A)
                      + (1/a) * (r sum_A (1 - cos k_A) - rho) ]

D_ov,tet = (rho/a) * (1 + gamma5 sign(H_tet)).
```

Here the active translation lattice is the rank-4 tetrahedral null-edge lattice
generated by four Hadamard/tetrahedral steps `h_A`. We are treating scalar
Wilson as a branch-mass input inside the overlap construction, not as a direct
chiral selector on the balanced origin kernel. The old scalar Wilson no-go still
applies to direct origin-kernel chirality selection, but we do not think it rules
out scalar Wilson inside the overlap sign-kernel route.

## Why D4 came up

The tetrahedral generators look closely related to the D4 root lattice. We asked
Aristotle to audit whether D4 should replace the tetrahedral lattice, act as an
envelope, or be treated as a red herring.

The Aristotle D4 job, C250, ran out of budget before producing a formal Lean file
or polished audit note. However, it did return a useful checkpoint. Please treat
the following as a partial external result that needs mathematical checking, not
as a settled theorem.

## C250 partial result from Aristotle

Let `Lambda_H` be the lattice generated by the four Hadamard/tetrahedral
generators. Let

```text
D4 = { x in Z^4 : sum_i x_i is even }.
```

Aristotle reported:

1. `Lambda_H <= D4`.

2. The index is:

```text
[D4 : Lambda_H] = 8.
```

3. Since the Hadamard matrix `H` has `H^2 = 4 I` and `|det H| = 16`, we have:

```text
[Z^4 : Lambda_H] = 16.
```

Because `[Z^4 : D4] = 2`, the tower gives `[D4 : Lambda_H] = 8`.

4. The quotient structures were reported as:

```text
Z^4 / Lambda_H ~= Z/4 x Z/2 x Z/2
D4 / Lambda_H ~= (Z/2)^3
```

5. A Lean-friendly characterization was suggested:

```text
Lambda_H = { x in Z^4 : H x = 0 mod 4 }.
```

6. Dual-lattice relation:

```text
Lambda_H^* = (1/4) Lambda_H
D4^* <= Lambda_H^*
[Lambda_H^* : D4^*] = 8
```

The interpretation offered was that the D4 Brillouin torus is an 8-fold folding
of the tetrahedral torus, so switching from `Lambda_H` to full D4 would collapse
or identify branch data. Therefore the C240 16-branch theorem on the
tetrahedral torus cannot be assumed to survive on the D4 torus.

7. Aristotle also reported that full D4 adds half-step composites represented
by terms like:

```text
(h_A + h_B)/2
(h_A - h_B)/2
```

The claim was that the half-sums are timelike and the half-differences are
spacelike under the physical Lorentzian interpretation, not null. Therefore full
D4 adds non-null primitive translations if used as the actual lattice.

8. The provisional recommendation was:

```text
D4 is an envelope / comparison object, not the Gate C1 release lattice.
Lambda_H remains the active null-edge translation lattice.
```

claim that the microscopic steps are lightlike?

9. If we later include D4 as an envelope or comparison object, what exact
theorems should we formalize first?

## Current strategic question

Our current tentative conclusion is:

```text
Use Lambda_H for Gate C1.
Use D4 only as an envelope, quotient/folding comparison, or later symmetry lane.
Do not replace Lambda_H by D4 unless we are prepared to redo the branch,
gap, no-mirror, and overlap proofs from scratch.
```

Is that the right call?

If not, please propose the strongest alternative. In particular, if D4 should
play a more central role, please explain exactly how it avoids damaging the
null-edge interpretation and how it would affect the overlap operator,
Brillouin-zone branch count, scalar Wilson branch masses, and no-mirror-pole
proof.

Please use both lattice-theory reasoning and lateral physics thinking. We are
especially interested in connections to known lattice-fermion constructions
such as overlap/Ginsparg-Wilson fermions, hyperdiamond or A4*/D4 lattices,
minimally doubled fermions, flavored mass terms, and any D4/F4/triality
constructions that might help.
