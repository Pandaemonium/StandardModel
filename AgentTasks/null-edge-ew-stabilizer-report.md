# Electroweak stabilizer kernel/rank — Lean report (T9, stage 1)

This implements **stage 1** of T9 (Working Plan §17.3 Theorem C, grand-strategy
§C.5 / §E.2): the finite-dimensional electroweak orbit-obstruction map has a
one-dimensional kernel (the electromagnetic `u(1)_em` line generated by `Q`) and
rank three. It is stated and labelled as a **structural theorem given the SM

throughout; no "gauge symmetry breaking" wording appears.

## Normalization conventions (documented in-file)

* **Generator convention: Hermitian.** The generators `T₁, T₂, T₃` are the Pauli
  matrices divided by two, and `Y/2 = (1/2)·I` is the Hermitian hypercharge
  generator. The physical anti-Hermitian Lie-algebra generators are
  `i T₁, …, i Y/2`; the factor `i` is an overall scalar that changes neither the
  kernel nor the rank, so the theorems hold verbatim in the anti-Hermitian
  convention as well.
* **Hypercharge:** `Y(H) = 1`, hence the hypercharge generator on the doublet is
  `Y/2 = (1/2)·I`.
* **Charge generator:** `Q = T₃ + Y/2 = diag(1, 0)` (`Qmat` in the file). Its
  coordinate vector in the basis `(T₁, T₂, T₃, Y/2)` is `Qgen = (0,0,1,1)`.
* **Vacuum section:** `H₀ = (0, v/√2)` with `v ≠ 0`.
* **Scalar field of the Lie algebra:** `su(2)_L ⊕ u(1)_Y` is the *real*
  4-dimensional Lie algebra with real coordinates `(x₀,x₁,x₂,x₃)` along
  `(T₁,T₂,T₃,Y/2)`. The orbit map is therefore an **ℝ-linear** map
  `B_EW : ℝ⁴ → ℂ²` (`ℂ²` carried as a real vector space of dimension 4). Using
  real coordinates is what makes the kernel one-dimensional; over ℂ-coordinates
  the count would differ, so this convention is essential and explicit.

In closed form, `B_EW(x) = (κ·(x₀ − i·x₁), κ·(x₃ − x₂))` with `κ = v/(2√2)`,
and this is certified equal to the matrix definition `X ↦ ρ(X) H₀`.

## Statements proved (all `sorry`-free)

Definitions: `T1, T2, T3, Yhalf, Qmat` (generators/charge as `2×2` complex
matrices), `H0` (vacuum), `rho` (representation of real coordinates), `Qgen`
(coordinate vector of `Q`), `kappa`, `B_EW` (the ℝ-linear obstruction map), and
`qform` (the quadratic obstruction `q(X) = ‖B_EW X‖²`).

* `Qmat_mulVec_H0` — `Q H₀ = 0` (the charge stabilises the vacuum).
* `rho_Qgen` — `ρ(Qgen) = Q`, i.e. `Qgen` represents `Q = T₃ + Y/2`.
* `B_EW_eq_matrix` — faithfulness: closed form `B_EW x = ρ(x) H₀`.
* `kappa_ne_zero`, `Qgen_ne_zero` — non-degeneracy helpers.
* **`ew_stabilizer_kernel`** — `ker B_EW = span ℝ {Q}` (the `u(1)_em` line).
* **`ew_stabilizer_kernel_finrank`** — `dim (ker B_EW) = 1`.
* **`ew_stabilizer_rank`** — `rank B_EW = dim (range B_EW) = 3` (via rank–nullity
  `4 = 1 + 3`).
* **`ew_quadratic_form_kernel`** — `{x | q(x) = 0} = span ℝ {Q}`; together with
  `qform_eq_zero_iff` (`q(x) = 0 ↔ B_EW x = 0`) this shows the quadratic form's
  kernel matches the linear map's, and its rank is likewise three.
