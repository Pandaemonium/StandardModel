## Program context

The null-edge program's P9 conjecture asks whether finite diamond bookkeeping
can separate:

- boundary-exact or internal bookkeeping, which should be invisible to closed
  bulk tests;
- visible Plucker/closure defects, which should source a bulk diamond response;
- a residual harmonic/noise channel, which is the only plausible finite analogue
  of a fluctuating Lambda source.

We have already proved toy finite algebra:

- boundary-exact sources vanish against closed tests;
- nonzero closed-test pairing witnesses visible residual source content;
- weighted diagonal noise responses obey finite bounds;
- an abstract self-adjoint projector makes harmonic tests see only the projected
  source;
- projected noise kernels preserve positive-semidefinite response.

The scientific risk is that all of this remains generic linear algebra. To
become physics, P9 needs an explicit finite diamond metric/projector and a
response/noise law whose dimension, trace, spectrum, and condition number depend
on causal-diamond geometry.
